# 悦动体育——PK乒乓球

  "悦动体育——PK乒乓球"是一款线上预约PK乒乓球和查看比赛约战的应用。

  主要功能:   
  
    1.PK分类：男单打、女单打、男双打、女双打等类型;
    
    2.首页：比赛约战、资讯、最新比赛、场馆精选;
    
    3.场馆：精选上海市的乒乓球场馆、以及场馆下的PK列表;

    4.价格：根据不同的PK选手进行收费，均以每场为单位计算单价;
    
    5.预约：全部PK、等待PK、已完成等预约订单;
    
    6.我的：我的消息、我的预约、我的收藏、相关设置;
